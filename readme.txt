Add a password file to the publication metabox to protect against accidental publication of drafts or updating of published post and pages

== Description ==

A simple plugin that adds a password to the publication metabox. This is to help avoid accidental publication.

Think of obituaries that you keep updated but you don't want to get published by mistake!!!

The password component is loaded onto the editor for posts and pages by default.

Use the filter "wp-password-protect-publication-supported_post_types" to adjust the supported post types.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wp-password-protect-publication` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
